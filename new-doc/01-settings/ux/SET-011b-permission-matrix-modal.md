interface PermissionMatrix {

## Accessibility

- **Table Semantics**: Proper `<table>`, `<thead>`, `<tbody>`, `<th scope>` markup
- **Row/Column Headers**: `scope="col"` for roles, `scope="row"` for modules
- **Cell Text**: Readable text alternative for permission codes
- **Keyboard Navigation**:
  - Tab to export buttons
  - Arrow keys to navigate table cells (when focused)
  - Enter/Space to activate buttons
- **Screen Reader**:
  - Announces "Permission matrix table, 10 roles by 40 modules"
  - Reads cell as "Production Manager, Production WO Execution, Full CRUD access"
  - Announces column/row context on cell focus
- **Contrast**:
  - CRUD cells: Green background (#10B981), white text (7:1)
  - RU cells: Blue background (#3B82F6), white text (6:1)
  - R cells: Gray background (#6B7280), white text (5:1)
  - - cells: Light gray background (#E5E7EB), dark text (8:1)
- **Focus Indicators**: 2px solid blue outline on keyboard focus
