## Goal

Comprehensive executive performance dashboard consolidating 10+ KPI cards, 6+ charts (OEE trends, downtime Pareto, machine status, energy, utilization heatmap, quality metrics) with real-time updates and drill-down capabilities.

---

## User Story

As an **Executive**, I want to **view all key production metrics on one dashboard** so that **I can quickly assess plant performance and identify issues**.

---

## Scope

**KPI Cards (10)**
- OEE Today, Availability, Performance, Quality
- Total Output, Good Output, Scrap %
- Downtime Today, Active Alerts, Energy Consumed

**Charts (6+)**
- OEE 7-day trend line chart
- Downtime Pareto chart
- Machine status grid
- Utilization heatmap (hourly, today)
- Energy consumption by machine
- Quality yield trend

**Features**
- Auto-refresh every 30 seconds
- Date range selector (Today, Week, Month)
- Line/Machine filter
- Export dashboard as PDF
- Drill-down from any chart to detail view

---

## Acceptance Criteria

```gherkin
Scenario: Dashboard loads with all widgets
  Given user navigates to /oee/dashboard
  When page loads
  Then displays:
    | Section | Widgets |
    | KPIs | 10 metric cards in grid |
    | Trends | OEE 7-day line chart |
    | Downtime | Pareto chart |
    | Machines | Status grid with live updates |
    | Energy | Bar chart by machine |
    | Heatmap | Utilization heatmap (hourly) |
  And all data loads within 2 seconds

Scenario: Auto-refresh
  Given dashboard open
  When 30 seconds pass
  Then all metrics refresh
  And new data highlighted
  And no full page reload

Scenario: Drill-down
  Given OEE trend chart
  When user clicks a data point (Jan 10)
  Then navigate to detailed view for that day
  And show hourly breakdown
```
