The Problem We're Solving

🔴 Current Process (Manual Excel)

  • ⏱ 30-60 min daily spent opening, copying, and formatting KPI data across multiple Excel files
  • 📄 3+ Excel files (80MB each) opened simultaneously — slow, crash-prone
  • 📋 Copy/paste between spreadsheets to build daily reports
  • 🔍 No quick drill-down — finding root cause of a yield issue requires manually filtering rows
  • 👥 No standard view for different roles — everyone sees raw data
  • 📅 Historical comparison requires manually opening old files
  • ⚠ Risk of formula errors and inconsistent calculations
➔

🟢 KPI Dashboard (Automated)

  • ⚡ Upload files once each morning — all dashboards update instantly (under 60 seconds)
  • 💻 Lightweight web app runs in your browser — no Excel needed to view KPIs
  • 📈 Pre-built charts, tables, and trend views — zero manual formatting
  • 🔎 Click any line → see SKU breakdown → see trends. Instant drill-down
  • 👥 Role-based views: Line Leaders, Supervisors, Management each get relevant data
  • 📅 52 weeks of history cached locally — compare weeks, periods, years instantly
  • ✓ Consistent calculations using 4-4-5 fiscal calendar — no formula mistakes

How It Works

1

Upload Excel Files

Drag & drop your daily Excel files into the sidebar — Line Yields, Prod Pallets, Shift Report, On-Hand Stock

2

Automatic Parsing

System reads all sheets, maps columns, validates data. Progress shown in real-time

3

Local Storage

Processed data cached in browser (IndexedDB). Line Yields: incremental append. Others: daily replace

4

Instant Dashboards

Navigate between 15+ interactive views — all KPIs, charts, and tables update automatically

Daily Workflow

⚠ Before (Current Process)

  1. 1 Open Line Yields (80MB) — wait 2-5 min for Excel to load
  2. 2 Open Prod Pallets — wait again
  3. 3 Manually filter dates, lines, teams in each file
  4. 4 Copy relevant data into summary spreadsheet
  5. 5 Calculate KPIs manually / update formulas
  6. 6 Format charts and tables for reporting
  7. 7 Email or share reports manually

⏱ ~45-60 minutes daily

✓ After (KPI Dashboard)

  1. 1 Open dashboard in browser (instant)
  2. 2 Drag & drop today's Excel files into sidebar
  3. 3 Wait 30-60 seconds for automatic parsing
  4. 4 All dashboards, charts, and KPIs are ready
  5. 5 Click through views — drill down as needed
  6. 6 Copy-to-clipboard for email, or print reports

⏱ ~2-5 minutes daily
