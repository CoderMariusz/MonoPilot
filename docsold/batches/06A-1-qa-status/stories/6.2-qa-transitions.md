# Story 6.2: QA Status Transition Rules

## Goal
Enforce valid QA status transitions.

## State Machine
- pending → passed, failed, quarantine
- quarantine → passed, failed
- failed → quarantine (only)
- passed → (no changes, Manager override only)

## Acceptance Criteria
- Invalid transition blocked with error message
- Valid transitions allowed
- Manager can override passed status
