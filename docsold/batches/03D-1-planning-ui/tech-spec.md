# Batch 03D-1: Planning UI - Technical Specification

**Batch ID:** 03D-1
**Epic:** 3 - Planning Operations
**Stories:** 3.23-3.30
**Prerequisites:** Batches 03A-1, 03B-1, 03C-2

## Overview

Planning module UI redesign with unified header, stats cards, dashboard tables, and responsive design.

---

## Components

### PlanningHeader (Story 3.23)
- Logo + Navigation tabs + Settings button
- Height: 60px
- Active tab: underline border-green-600

### StatsCards (Story 3.24)
- Top cards showing key metrics
- PO/TO/WO counts by status

### PurchaseOrdersDashboard (Story 3.25)
- Recent POs table
- Status badges with colors

### PurchaseOrderDetails (Story 3.26)
- PO detail page with tabs
- Lines, Approval, Activity tabs

### WorkOrdersTable (Story 3.27)
- WO list with status, product, quantity
- Quick actions

### TransferOrdersTable (Story 3.28)
- TO list with from/to warehouse
- Ship/Receive status

### MobileResponsive (Story 3.29)
- Stack layout on mobile
- Collapsible sidebar

### ColorConsistency (Story 3.30)
- Unified color palette
- Status badge colors
