# Story 4.21: Genealogy Tree View

**Epic:** 4 - Production Execution | **Status:** done | **Priority:** P1 | **Story Points:** 2 | **Effort:** 1 day | **Completed:** 2025-12-07

> **Note:** This story focuses on the UI display of genealogy. Core genealogy logic is in **Story 4.19** (04B-2).

## User Story

**As a** Quality Manager
**I want** to view genealogy tree for any LP
**So that** I can trace products forward and backward for recalls

## Acceptance Criteria

### AC-4.21.1: Forward Tracing View
**Given** user views LP details
**When** clicks "Trace Forward"
**Then** shows all child LPs produced from this LP:
- LP → WO → Output LPs
- Shows consumed_qty per link
- Status: active/reversed

### AC-4.21.2: Backward Tracing View
**When** clicks "Trace Backward"
**Then** shows all parent LPs that went into this LP:
- Output LP → WO → Input LPs
- Shows consumed_qty per link
- Status: active/reversed

### AC-4.21.3: Multi-Level Tracing
**Then** genealogy tree shows multiple levels:
- RM → WIP → FG → Customer
- Expandable tree nodes
- Depth limit configurable

### AC-4.21.4: Reversed Link Display
**When** genealogy link is reversed
**Then** shows:
- [REVERSED] badge
- Reversed by / when / reason
- Strikethrough visual

### AC-4.21.5: FDA Compliance
**Then** Genealogy view supports FDA 21 CFR Part 117 traceability requirements
