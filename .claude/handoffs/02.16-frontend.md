# Handoff: 02.16 Product Advanced Features - Frontend

## Story
4 features: Image upload, Product clone, Barcode, Categories + Tags.

## Components to Create

### In `apps/frontend/components/technical/products/`:
1. `ProductImageUpload.tsx` - Drag-drop zone, preview, remove button, progress
2. `ProductImagePreview.tsx` - Full-size image with click-to-zoom
3. `ProductCloneModal.tsx` - Modal with prefilled fields, SKU suggestion, options checkboxes
4. `ProductBarcodeModal.tsx` - Format selector (Code128/EAN-13), preview, print/download
5. `BarcodeDisplay.tsx` - Inline barcode image with print action
6. `ProductCategorySelect.tsx` - Hierarchical dropdown (tree view)
7. `ProductTagsSelect.tsx` - Multi-select with autocomplete, inline create
8. `ProductTagBadge.tsx` - Color badge (max 3 visible + "+N more")

### In `apps/frontend/components/settings/`:
9. `CategoryTreeView.tsx` - Drag-drop tree for category management
10. `CategoryModal.tsx` - Create/edit category (name, parent, description)
11. `TagsDataTable.tsx` - Tags list with usage counts
12. `TagModal.tsx` - Create/edit tag (name, color picker)

### Pages
- Extend: `apps/frontend/app/(authenticated)/technical/products/page.tsx` (add image column, clone action)
- New: `apps/frontend/app/(authenticated)/settings/product-categories/page.tsx`
- New: `apps/frontend/app/(authenticated)/settings/product-tags/page.tsx`

## Clone Modal Spec
- SKU field: prefilled "{original}-COPY", real-time uniqueness validation
- Name field: prefilled, editable
- Checkboxes: [x] Include allergens, [x] Include categories/tags, [ ] Include image
- Button: "Clone Product" -> toast success -> close modal -> refresh list
