### Keyboard Navigation
- **Tab**: Navigate between form fields, buttons
- **Enter**: Select radio button (for strategy options: selects strategy and auto-advances to Step 3), activate button
- **Arrow Keys**: Navigate table rows and strategy option radios
- **Escape**: Close wizard (with confirmation)

**Step 2 Strategy Selection Detail**: Arrow Keys to navigate strategy options, Enter to select and auto-advance to Step 3, or Tab to Next button for manual advance.
