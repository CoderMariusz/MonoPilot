## Key Components

### 1. WO Summary Card
- WO number, status, product, batch
- Quantity progress (planned, output, remaining)
- Visual progress bar
- All fields read-only

### 2. Yield Summary Card
- Overall yield with target comparison
- Output yield (actual vs planned)
- Material yield (consumed vs planned)
- Operation yield (from operations)
- Trend chart (last 7 days)
- Color coding: ğŸŸ¢ â‰¥95% | ğŸŸ¡ 80-94% | ğŸ”´ <80%
- Link to Material Consumption view

### 3. Output History Table
- Columns: LP Number, Qty, Batch, QA Status, Location, Expiry, Created, Actions
- Filters: QA Status, Location, Date
- Sort: Created At (desc)
- Actions: [View] [Label]
- Summary section: Total outputs, qty, status breakdown
- Pagination

### 4. By-Products Section
- Auto-create toggle status
- Table: Product, Expected Qty, Actual Qty, Status, Actions
- Expected vs actual yield comparison
- Actions: [View LPs] [Add More] [Register Now]
- [Register All By-Products] batch action
- Genealogy info note

### 5. Register Output Modal
- Pre-filled product info (read-only)
- Form fields: Quantity, UoM, Batch, QA Status, Location, Expiry, Notes
- Auto-calculations: Expiry date, batch number
- Output summary preview
- After registration preview
- Validation: Real-time, inline errors

## States (All 4 Required States)

### 1. Loading State
- Skeleton loaders for WO Summary, Yield Summary, Output History, By-Products
- Text: "Loading output history...", "Loading by-products..."
- No interaction possible during load

### 2. Empty State (No Outputs Registered)
- WO Summary shows 0% progress
- Yield Summary shows "N/A"
- Output History: Empty state illustration with prompt
  - "No outputs registered yet"
  - "Register your first output to start tracking production progress"
  - [Register First Output] button
  - Alternative: [Switch to Scanner View]
- By-Products: Info message
