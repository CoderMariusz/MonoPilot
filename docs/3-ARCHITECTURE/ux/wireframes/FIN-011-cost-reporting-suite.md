## Key Components

1. **Report Type Selector** - Switch between 4 report types
   - **Cost by Product**: Product-level cost analysis with variance
   - **Cost by Period**: Time series cost trends (daily/weekly/monthly/quarterly)
   - **Cost by Production Line**: Line-level cost comparison with efficiency metrics
   - **Cost by Cost Center**: Cost center allocation with budget variance

2. **KPI Cards (4 cards per report type)** - Summary metrics
   - **Cost by Product KPIs**: Total Cost, Total Units, Avg Cost/Unit, Variance
   - **Cost by Period KPIs**: Total Cost (period), Avg Period Cost, Max Period Cost, Period-over-Period Change
   - **Filters**: Date range, category, product, line, cost center, variance threshold

3. **Cost Breakdown Table** - Detailed cost data
   - **Cost by Product Table**: Rank, SKU, Product, Units, Material/Labor/Overhead, Total, $/Unit, Variance %
   - **Cost by Period Table**: Period, Material/Labor/Overhead, Total, vs Prev %, Cumulative, Units
   - **Sortable columns**: Click column header to sort
   - **Drill-down**: Click product/period â†’ detailed analysis
   - **Pagination**: 10 per page (desktop/tablet), 5 per page (mobile with "Load More")

4. **Cost Trend Chart** - Visual trend analysis
   - **Cost by Product**: Multi-line chart (Material, Labor, Overhead, Total) over time
   - **Cost by Period**: Line chart with moving average, period-over-period change indicators
   - **Time Range**: Configurable (weekly, monthly, quarterly)
   - **Key Insights**: Trend summary, average increase/decrease, recommendations

5. **Top Contributors Panel** - Largest cost drivers
   - **Top Cost Products**: Top 3-5 products by total cost, % contribution, key drivers
   - **Action Links**: View product detail, variance analysis, drill-down

6. **Export & Actions** - Report distribution
   - **Export to Excel**: Formatted workbook with charts (multiple sheets: Summary, Data, Charts)
   - **Export to CSV**: Raw data for analysis
   - **Export to PDF**: Print-friendly report with branding
   - **Email Report**: Send to finance/production team
   - **Schedule Report**: Automated recurring report generation

---

## Main Actions

### Primary
- **Switch Report Type** - Select report type (Product, Period, Line, Cost Center)
- **Apply Filters** - Filter by date range, category, product, line, cost center, variance threshold
- **Export Report** - Download to CSV/Excel/PDF
- **View Product/Period Detail** - Click product/period â†’ Navigate to detailed analysis

### Secondary
- **Email Report** - Send report to finance/production team via email
- **Schedule Report** - Configure automated report generation (daily/weekly/monthly)
- **Drill Down** - Navigate to variance analysis, work order detail, cost breakdown
- **Print Report** - Print-friendly view

### Filters
- **Date Range Filter**: Date picker (Last 7 days, Last 30 days, Last 90 days, Custom)
- **Period Type Filter**: Dropdown (Daily, Weekly, Monthly, Quarterly) - for Cost by Period report
- **Product Category Filter**: Dropdown (All, or specific category)
- **Production Line Filter**: Dropdown (All, or specific line) - for Cost by Line report
- **Cost Center Filter**: Dropdown (All, or specific cost center) - for Cost by Cost Center report
- **Variance Threshold Filter**: Input (only show items with variance > X%)

### Export
- **Export to CSV** - Download raw cost data
- **Export to Excel** - Download formatted report with multiple sheets (Summary, Detailed Data, Charts)
- **Export to PDF** - Download print-friendly report with company branding

## States

- **Loading**: Skeleton cards (4 KPI cards) + skeleton table + skeleton chart, "Loading cost report data..." text
- **Success**: Populated 4 KPI cards, cost breakdown table (paginated), trend chart, top contributors panel
- **Empty**: "No cost data available yet" message, Quick actions to create work order / set up standard costs, Finance Reporting Guide link
- **Error**: "Failed to load cost report" warning, Retry + Contact Support buttons, Quick actions still available

## Business Rules

### Cost by Product Report
- **Default View**: Last 30 days, all products, ordered by total cost DESC
- **Cost per Unit**: total_cost / units_produced
- **Variance**: total_cost - (standard_cost_per_unit Ã— units_produced)
- **Variance %**: (variance / (standard_cost_per_unit Ã— units_produced)) Ã— 100

### Cost by Period Report
- **Period Types**: Daily (calendar day), Weekly (Mon-Sun), Monthly (calendar month), Quarterly (Q1-Q4)
- **Period-over-Period Change**: ((current_period - previous_period) / previous_period) Ã— 100
- **Cumulative Cost**: Running SUM(period_cost) from start date
- **Moving Average**: Calculate 3-period or 7-period moving average for trend smoothing

### Export
- **Excel Format**: Multiple sheets (Summary, Detailed Data, Charts), formatted headers, currency formatting
- **PDF Format**: Print-friendly layout, page breaks, company branding (logo, name), header/footer with page numbers
- **CSV Format**: Raw data with proper escaping
