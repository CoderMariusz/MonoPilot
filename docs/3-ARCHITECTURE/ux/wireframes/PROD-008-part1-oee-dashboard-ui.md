## Key Components

1. **Current Shift Info Panel** - Real-time shift status
   - **Shift Name**: Display current shift name (Day, Afternoon, Night) from shift_management table
   - **Time Display**: Shift start-end time (e.g., 06:00-14:00), current time, elapsed time
   - **Duration**: Gross duration (e.g., 480 min) and net production time (e.g., 450 min after breaks)
   - **Machine/Line Status**: Count of active machines vs total, active lines vs total
   - **View All Button**: Navigate to shift schedule/configuration
   - **Status Indicator**: Active (green) / Ended (gray)

2. **OEE Gauges (4 circular gauges)** - Visual OEE metrics
   - **Availability Gauge**:
     - Formula: (Operating Time / Planned Time) Ã— 100
     - Display: 87.5% (circular gauge with color coding)
     - Color: Green >= 85%, Yellow 70-84%, Red < 70%
     - Details: Operating time (420 min), Planned time (480 min), Downtime (60 min)
     - Target comparison: Green if >= target, red if < target
   - **Performance Gauge**:
     - Formula: (Actual Output / Theoretical Output) Ã— 100
     - Display: 90.0%
     - Color: Same as Availability
     - Details: Actual output (900 pcs), Theoretical output (1000 pcs), Cycle time (28s/unit)
     - Target comparison: Green if >= target, red if < target
   - **Quality Gauge**:
     - Formula: (Good Output / Total Output) Ã— 100
     - Display: 95.0%
     - Color: Same as Availability
     - Details: Good output (950 pcs), Total output (1000 pcs), Defect rate (5%)
     - Target comparison: Green if >= target, red if < target
   - **Overall OEE Gauge**:
     - Formula: Availability Ã— Performance Ã— Quality
     - Display: 74.8% (calculated as 0.875 Ã— 0.9 Ã— 0.95 = 0.748)
     - Color: Red if < target_oee_percent, Green if >= target_oee_percent
     - Target line: 85% (from settings.target_oee_percent)
     - Breakdown: Show calculation formula (87.5% Ã— 90% Ã— 95% = 74.8%)
     - [View Breakdown] button â†’ drills down to A/P/Q details

3. **OEE Trend Chart** - Historical OEE performance
   - **Chart Type**: Line chart (last 7 days OEE %)
   - **Y-axis**: OEE % (0-100%)
   - **X-axis**: Dates (MM-DD format)
   - **Target Line**: Horizontal line at 85% (dotted)
   - **Data Points**: Daily average OEE % with clickable markers
   - **Stats Display**: Avg OEE, Best day (date + value), Worst day (date + value)
   - **Filters**:
     - Date range dropdown (Last 7 Days, Last 30 Days, Custom Range)
     - Machine filter (All, or specific machine)
     - Shift filter (All, Day, Afternoon, Night)
   - **Actions**:
     - [Drill Down] button â†’ navigate to daily detail view
     - [Export CSV] button â†’ download trend data

4. **Machine Comparison Table** - Per-machine OEE breakdown
   - **Columns**: Machine, Line, Current WO, Availability, Performance, Quality, OEE, Status, Actions
   - **Row Details**:
     - Line 1: Machine name, line, WO number, A/P/Q/OEE values with color coding, status badge
     - Line 2: Downtime summary (duration + category), Output progress (actual/planned), Defects count
   - **Color Coding**:
     - Red: < 70% (poor)
     - Yellow: 70-84% (acceptable)
     - Green: >= 85% (good)
