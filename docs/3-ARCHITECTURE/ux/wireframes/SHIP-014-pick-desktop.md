## Focus State Specifications

### Focus Ring Styling
- **Focus Ring Color**: 2px solid #2563eb (blue-600)
- **Focus Ring Offset**: 2px from element border
- **Focus Ring Opacity**: 100% (fully opaque)
- **Focus Ring Animation**: None (instant appearance)
- **Focus Persistence**: Visible until focus moves away

### Interactive Element Focus States

#### Buttons (All Variants)
- **Background**: Maintain original color
- **Focus Ring**: 2px solid #2563eb, offset 2px
- **Outline**: None (use ring instead)
- **Cursor**: pointer
- **Keyboard Activation**: Enter or Space
- **Example**: [âœ“ Confirm Pick], [Cancel], [Complete Pick List]

#### Table Rows
- **Focus Target**: Entire row (clickable)
- **Focus Ring**: 2px solid #2563eb around full row width
- **Focus Ring Offset**: 1px from row edges
- **Row Background**: Maintain striped pattern or hover color
- **Accessibility Label**: aria-label includes line number and product name
- **Keyboard Navigation**: Arrow Up/Down to navigate rows, Enter to select

#### Dropdown Selectors
- **Focus Ring**: 2px solid #2563eb around dropdown box
- **Focus Offset**: 2px
- **Open Indicator**: Arrow changes to indicate focused state
- **Keyboard Access**: Tab to focus, Down/Up arrows to navigate options, Enter to select

#### Modal Dialogs
- **Focus Trap**: Initially focus first focusable element (e.g., first text input or button)
- **Focus Ring**: Applied to all interactive elements within modal
- **Trap Behavior**: Tab loops within modal (last element Tab goes to first)
- **Escape Handling**: Escape key closes modal
- **Return Focus**: After modal closes, focus returns to element that opened it
- **Example**: Pick Confirmation Modal focuses on first LP selection button

#### Input Fields (Quantity, Manager Code, Notes)
- **Focus Ring**: 2px solid #2563eb around input border
- **Focus Offset**: 2px
- **Background**: Light background on focus (e.g., #f0f9ff)
- **Cursor**: text (for text inputs), default (for number spinners)
- **Keyboard Support**: Standard text input behavior

#### Checkboxes (Allergen Overrides)
- **Focus Ring**: 2px solid #2563eb around checkbox box
- **Focus Offset**: 2px
- **Check Indicator**: Visible when checked (checkmark icon)
- **Keyboard Activation**: Space to toggle when focused
- **Label Association**: aria-label on checkbox or linked <label>

### Skip Link Implementation

#### Skip to Content Link
- **Display**: Hidden by default (`display: none` or `clip: rect(0 0 0 0)`)
- **Trigger**: Appears on first Tab press (`:focus` pseudo-class)
- **Text**: "Skip to main content" or "Skip to pick lines"
- **Focus Ring**: 2px solid #2563eb
- **Target**: Jump to pick lines table (id="pick-lines-table")
- **Keyboard**: Tab to activate, Enter to jump

#### Skip Link CSS Example
```css
.skip-link {
  position: absolute;
  top: -40px;
  left: 0;
  background: #2563eb;
  color: white;

  top: 0;
  outline: 2px solid #1e40af;
  outline-offset: 2px;
}
```
