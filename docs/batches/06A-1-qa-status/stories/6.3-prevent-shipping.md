# Story 6.3: Prevent Shipping Non-Passed LPs

## Goal
Block shipping of non-passed LPs to ensure only approved inventory ships.

## Acceptance Criteria
- LP with qa_status != 'passed' blocked from shipping (SO, TO)
- Error shown: "LP must have QA status: passed"
- Only passed LPs shown in picker
