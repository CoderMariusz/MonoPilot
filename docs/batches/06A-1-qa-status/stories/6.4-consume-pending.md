# Story 6.4: Control Consumption of Pending LPs

## Goal
Allow Admin to configure if pending LPs can be consumed in production.

## Acceptance Criteria
- Toggle: quality_settings.allow_consume_pending
- ON: production can consume pending LPs (with warning)
- OFF: only passed LPs can be consumed
- Default = OFF (strict mode)

## Technical
- FR: QC-FR-04
- Prerequisites: Story 6.1, Epic 4
