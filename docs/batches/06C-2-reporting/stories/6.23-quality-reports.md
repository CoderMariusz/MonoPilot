# Story 6.23: Quality Reports Generation

## Goal
Generate quality reports for stakeholders.

## Report Types
- NCR Summary (date range, severity, type)
- Test Results Summary (product, date)
- Quality Hold Report
- CoA Compliance Report
- Supplier Quality Performance

## Acceptance Criteria
- Preview report before export
- Filter parameters per report type

## Technical
- FR: QC-FR-25
- API: POST /api/quality/reports/generate
- Prerequisites: All Epic 6 stories
