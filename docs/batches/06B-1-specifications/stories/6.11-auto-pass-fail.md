# Story 6.11: Auto-Calculate Pass/Fail for Numeric Tests

## Goal
Auto-determine pass/fail for numeric tests based on spec limits.

## Acceptance Criteria
- If result within min/max → passed
- If result outside range → failed
- Variance calculated: (result - expected) / expected × 100
